升级到 OS X Yosemite，

于 Mac App Store 免费提供。

1. 查看你的 Mac 硬件

如需查看 Mac 机型的详细信息，请点击屏幕左上角的 Apple 图标，选择 "关于本机"，然后点击 "更多信息"。以下 Mac 机型与 OS X Yosemite 兼容：

  • iMac (2007 年中期或之后的机型)
  • MacBook (2008 年后期的铝金属机型或 2009 年前期或之后的机型)
  • MacBook Pro (2007 年中期/后期或之后的机型)
  • MacBook Air (2008 年后期或之后的机型)
  • Mac mini (2009 年前期或之后的机型)
  • Mac Pro (2008 年前期或之后的机型)
  • Xserve (2009 年前期)

2. 查看你的 OS X 版本

将 "关于本机" 窗口打开后，查看 Mac 运行的是哪个版本的 OS X，你将看到运行的版本号就位于序列号下方。你可从以下版本升级到 OS X Yosemite：

  • OS X Snow Leopard (10.6.8)
  • OS X Lion (10.7)
  • OS X Mountain Lion (10.8)
  • OS X Mavericks (10.9)

拥有较早版本的 OS X？了解如何升级 >

3. 免费升级到 OS X Yosemite

访问 Mac App Store 中的 OS X Yosemite 页面。点击 "下载" 按钮，然后按照屏幕说明开始升级。如果你没有宽带连接，你可以前往任意一家 Apple Store 零售店升级你的 Mac。

基本要求

  • OS X 10.6.8 或更新版本
  • 2GB 内存
  • 8GB 可用磁盘空间
  • 部分功能要求使用 Apple ID；须遵守服务条款。
  • 某些功能需要兼容的互联网服务供应商；可能需要付费。

功能要求

Handoff 和 Instant Hotspot

以下 Mac 机型支持这两项功能：

  • MacBook Air (2012 年或之后的机型)
  • MacBook Pro (2012 年或之后的机型)
  • iMac (2012 年或之后的机型)
  • Mac mini (2012 年或之后的机型)
  • Mac Pro (2013 年后期的机型)

Handoff 功能要求使用配备 Lightning 接口和安装 iOS 8 的 iPhone、iPad 或 iPod touch。

Instant Hotspot 功能要求使用配备蜂窝网络连接以及 Lightning 接口和安装 iOS 8.1 的 iPhone 或 iPad。要求使用运营商提供的个人热点服务。

通话功能

通话功能要求使用安装 iOS 8 的 iPhone。要求使用激活的运营商资费计划。

SMS

SMS 要求使用安装 iOS 8.1 的 iPhone。要求使用激活的运营商资费计划。

AirDrop

以下 Mac 机型支持 Mac 电脑和 iOS 设备之间的 AirDrop 功能：

  • MacBook Air (2012 年或之后的机型)
  • MacBook Pro (2012 年或之后的机型)
  • iMac (2012 年或之后的机型)
  • Mac mini (2012 年或之后的机型)
  • Mac Pro (2013 年后期的机型)

在 iOS 设备上使用 AirDrop 功能，要求使用配备 Lightning 接口并安装 iOS 7 或更新版本的 iPhone、iPad 或 iPod touch。

以下 Mac 机型支持在两台 Mac 电脑之间使用 AirDrop 功能：

  • MacBookPro (2008 年后期或之后的机型)*
  • MacBook Air (2010 年后期或之后的机型)
  • MacBook (2008 年后期或之后的机型)*
  • iMac (2009 年前期或之后的机型)
  • Mac mini (2010 年中期或之后的机型)
  • Mac Pro (2009 年前期推出配备 AirPort Extreme 卡的机型，以及 2010 年中期或之后的机型)

Time Machine

需要一个额外的硬盘或 AirPort Time Capsule (需单独购买)。

Photo Booth

须配备 FaceTime 或 iSight 摄像头 (内置或外置)、USB 视频类 (UVC) 摄像头或 FireWire DV 摄像机。如需在使用 DV 摄像机时实现背景特效，须配备定焦、曝光及白平衡功能。

FaceTime

视频通话要求使用内置的 FaceTime 摄像头、iSight 摄像头 (内置或外置)、USB 视频类 (UVC) 摄像头或 FireWire DV 摄像机，以及 128-Kbps 上行和下行互联网连接。进行 HD 高清视频通话须使用内置的 FaceTime HD 高清摄像头和 1-Mbps 上行和下行互联网连接。接收 HD 高清视频通话须使用支持此功能且配备 Intel 处理器的 Mac。了解更多

Boot Camp

支持现有通过 Boot Camp 分区安装的 Windows XP Service Pack 2、Windows Vista 或 Windows 7。安装新版 Boot Camp 须使用 Windows 7 或 Windows 8 (需单独购买)。

Exchange 支持

要求使用 Microsoft Exchange Server 2007 Service Pack 1 Update Rollup 4、Exchange Server 2010 或 Exchange 2013。自动设置要求启用 Microsoft Exchange Server 上的 Autodiscovery 功能。

AirPlay 镜像

要求使用 Apple TV (第二代或更新版本)。以下 Mac 机型支持此功能：

  • MacBook Pro (2011 年前期或之后的机型)
  • MacBook Air (2011 年中期或之后的机型)
  • iMac (2011 年中期或之后的机型)
  • Mac mini (2011 年中期或之后的机型)
  • Mac Pro (2013 年后期的机型)

点对点 AirPlay

要求使用配备 OS X Yosemite 的 Mac (2012 年或之后的机型) 和配备 Apple TV 软件 7.0 或更新版本的 Apple TV (第三代 rev A，A1469 型号或更新机型)。

Power Nap

以下 Mac 机型支持此功能：

  • 配备 Retina 显示屏的 MacBook Pro (2012 年中期或之后的机型)
  • MacBook Air (2010 年后期或之后的机型)
  • iMac (2012 年后期或之后的机型)
  • Mac mini (2012 年后期或之后的机型)
  • Mac Pro (2013 年后期的机型)

手势

要求使用 Multi‑Touch 触控板、Magic Trackpad 或 Magic Mouse。

VoiceOver 手势

要求使用 Multi‑Touch 触控板或 Magic Trackpad。

听写功能

要求使用麦克风 (内置或外置)。

Mac App Store

仅向美国和其他许多国家年满 13 周岁的用户提供。要求使用兼容的硬件、软件与互联网接入，建议使用宽带网络 (可能需要付费)。须遵守服务条款。
